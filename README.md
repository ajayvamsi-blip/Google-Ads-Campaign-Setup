# Google Ads Campaign Setup Project

## Project Overview
This project demonstrates the setup of a Google Search Ads Campaign for an online clothing store.

## Objective
Increase website traffic and generate sales through targeted Google Ads campaigns.

## Tools Used
- Google Ads
- Google Keyword Planner
- Microsoft Excel

## Campaign Goal
Website Traffic

## Target Audience
- Age: 18-35
- Location: India
- Interests: Fashion and Online Shopping

## Tasks Performed
- Keyword Research
- Campaign Structure Design
- Ad Group Creation
- Ad Copy Writing
- Budget Planning

## Expected Outcome
Increase website visitors, improve brand awareness, and drive online sales.
